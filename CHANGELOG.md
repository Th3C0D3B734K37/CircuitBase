# Changelog

All notable changes to CircuitBase will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Comprehensive component database with 90+ electronic components
- Datasheet integration with direct manufacturer links
- Multi-retailer purchase links (Robu, Amazon, AliExpress)
- Inventory management system
- Bill of Materials (BOM) creation and management
- Component comparison tool (up to 3 components)
- Smart search and filtering by category, tier, and interface
- Recently viewed components tracking
- Professional project documentation and setup

### Changed
- Complete UI redesign with modern, responsive interface
- Improved project structure for better maintainability
- Enhanced TypeScript type safety
- Optimized component performance

### Fixed
- Component data integrity and validation
- Link verification for datasheets and purchase options
- Responsive design issues across devices

## [1.0.0] - 2024-03-16

### Added
- Initial release of CircuitBase
- Component database with essential electronic components
- Basic inventory management
- Simple component search functionality
- Component detail views with specifications

## Version History

### Versioning Scheme
- **Major (X.0.0)**: Breaking changes, major features
- **Minor (X.Y.0)**: New features, improvements
- **Patch (X.Y.Z)**: Bug fixes, small improvements

### Release Types
- **Added**: New features and functionality
- **Changed**: Modifications to existing features
- **Deprecated**: Features that will be removed in future
- **Removed**: Features that have been removed
- **Fixed**: Bug fixes and issue resolutions
- **Security**: Vulnerability fixes and security improvements
